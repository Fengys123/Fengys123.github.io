题目要求

要求：给定n个重量为w1，w2…wn，价值为v1,v2….vn的物品，和容量为c的背包，求这个物品中一个最有价值的子集，使得满足背包容量的前提下，包内价值最大，注意：每个物品只能使用一次。
​

一、递归思路：自顶向下思考。

按照往常写出递归公式，终止条件和代码。

  • 递归公式：f(index, c) = Math.max( f(index-1, c), v[index] + f(index-1, c-w[index]) );

  • 终止条件：if(index < 0 || c <= 0) return 0;
