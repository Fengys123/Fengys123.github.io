TTCP简介

  • TTCP是一种应用层协议。
  • 主要用途：检测TCP连接的吞吐量。
  • 如何实现：客户端向服务端发送一个包，服务端收到返回确认信息（ack），当客户端收到服务端确认信息后，继续发送….. 循环指定次数之后，由程序计算出带宽。公式：带宽 = length * number / time。
